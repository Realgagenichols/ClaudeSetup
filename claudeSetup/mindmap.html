Project Lifecycle Flow
1
/new-project
Scaffold from templates, seed lessons, write initial SPEC.md
TEMPLATES SPEC.md SEED LESSONS
2
/brainstorm
Read lessons, explore design, update SPEC.md with RFC 2119 + scenarios
LESSONS.MD RFC 2119 GIVEN/WHEN/THEN
3
/plan
Hierarchical tasks in todo.md, every task traces to a requirement ID
tasks/todo.md R-ID TRACING
4
/implement
Subagent per task, re-reads lessons before each dispatch, TDD, per-task reviews
SUBAGENTS TDD READS LESSONS SPEC REVIEW
5
/review
3-stage: spec compliance, /reviewer agent (Opus), /review --fix with auto-repair
STAGE 1: SPEC STAGE 2: OPUS STAGE 3: FIX
6
Verify
Evidence before claims, run proof, capture lessons, promote patterns
VERIFICATION LESSONS CAPTURE
7
Ship
Merge, PR, or keep branch; promote cross-cutting lessons
FINISH BRANCH PROMOTE LESSONS
